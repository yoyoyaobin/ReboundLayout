# ReboundLayout
越界回弹效果的Layout

![image](https://github.com/yoyoyaobin/ReboundLayout/blob/master/app/src/main/assets/1.gif)

## 使用方式1(普通布局)
```
<com.hyb.rebound.ReboundLayout>
  <layout>
  </layout>
  </com.hyb.rebound.ReboundLayout>
```

## 使用方式2（需要scrollview效果的布局）
```
<com.hyb.rebound.ReboundScrollView>
  <com.hyb.rebound.ReboundLayout>
    <layout>
    </layout>
  </com.hyb.rebound.ReboundLayout>
</com.hyb.rebound.ReboundScrollView>
```
